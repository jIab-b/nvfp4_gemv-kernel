`copysign`

Copy sign of one input to another.

Syntax

```
copysign.type  d, a, b;



.type = { .f32, .f64 };
```

Description

Copy sign bit of `a` into value of `b`, and return the result as `d`.

PTX ISA Notes

Introduced in PTX ISA version 2.0.

Target ISA Notes

Requires `sm_20` or higher.

Examples

```
copysign.f32  x, y, z;

copysign.f64  A, B, C;
```
